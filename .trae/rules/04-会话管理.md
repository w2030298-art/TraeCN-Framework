# 会话管理

## 会话生命周期
会话开始 → 加载记忆 → 任务执行 → 保存状态 → 会话结束

## 会话开始（自动执行）
1. 读取 user_scope.md（用户画像）
2. 读取 project_scope.md（项目状态）
3. 检查 session-handoff.md（未完成任务）
4. 存在则询问："上次未完成 [任务]，是否继续？"

## 任务执行中
- 使用 TodoWrite 维护任务列表
- 重要决策立即写入 knowledge/decisions.md
- 用户说"暂停"→ 标记状态为"暂停"

## 会话结束
用户说"先到这里"/"下次继续"时：
1. 调用 Skill{name="session-handoff"}
2. 生成 session-handoff.md
3. 更新 project_scope.md

## 会话恢复
新会话检测到 session-handoff.md 时：
- 告知用户上次未完成工作
- 用户确认→恢复任务状态继续执行
- 用户拒绝→询问是否开始新任务
