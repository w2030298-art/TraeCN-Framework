# TraeCN-Framework

> 专为 Trae IDE 打造的 AI 编程助手框架。基于 Read/Write 工具的结构化记忆 + Skill 技能调度。

## 核心理念

传统方式用纯文本文件和庞大 Prompt 维护记忆和角色，存在记忆污染、多角色冲突、效率低下等问题。

本框架的方案：
- **结构化记忆**：Markdown 表格文件，易读、易更新、易搜索
- **Skill 技能调度**：通过 `/指令` 触发对应 Skill，任务隔离
- **Coordinator 模式**：主 Agent 负责任务拆解与调度，子任务委派给 Skill

## 目录结构

```
项目根目录/
├── .trae/              # 框架文件（保留）
│   ├── rules/          # 规则
│   └── skills/         # 技能
├── Core/记忆/          # 记忆文件（用户数据）
├── Builds/             # 已完成项目
├── Skills/             # 自定义技能
├── Evolve/             # 进化日志
└── Backup/             # 备份
```

## 快速开始

### 1. 导入框架
将 `.trae/` 文件夹复制到你的项目根目录。

### 2. 初始化
在 Trae IDE 中打开工作区，在聊天框输入：
```
/初始化
```

AI 助手将引导你完成：
- 创建目录结构
- 生成空白记忆文件
- 配置 Agent 名称

### 3. 使用

通过 `/指令` 触发技能：

| 指令 | 功能 |
|------|------|
| `/初始化` | 新用户首次使用 |
| `/重置` | 清除用户数据（需确认） |
| `/分析` | 需求分析 |
| `/设计` | 架构设计 |
| `/测试` | 质量验证 |
| `/部署` | 项目部署 |
| `/复盘` | 闭环复盘 |
| `/结束会话` | 会话交接 |

### 4. 重置

如需清除所有用户数据，输入：
```
/重置
```

系统会要求输入 Agent 名称确认，然后删除：
- `Core/` - 记忆文件
- `Builds/` - 已完成项目
- `Skills/` - 自定义技能
- `Evolve/` - 进化记录
- `Backup/` - 备份

保留：`.trae/` 框架文件

## 自定义

### 修改 Agent 名称
编辑 `.trae/rules/01-小T身份.md`，将 `小T` 替换为你的名字

### 添加自定义 Skill
1. 在 `.trae/skills/` 创建新目录
2. 创建 `SKILL.md`（参考 template/SKILL_TEMPLATE.md）
